#cpp-rails-chat-app

Rails + Docker + C++ + etension によるチャットアプリ開発リポジトリです。

## 予定機能

- チャットルーム作成・一覧・詳細
- メッセージ送信・一覧表示
- C++によるモデレーション
- C++による全文検索
- AIによる会話要約
- AIによる返信候補生成

## 開発環境

- macOS
- Docker / Docker Compose
- Rails
- PostgreSQL
- C++ Ruby Extension

## 開発方針

- 小さな単位で実装
- 動いたらすぐ commit
- 区切りごとに GitHub へ push
